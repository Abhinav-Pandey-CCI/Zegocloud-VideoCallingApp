#ZEGOCLOUD Video Calling App

A simple web-based video calling application built using ZEGOCLOUD UIKit. Supports real-time video, audio toggle, room join/leave, and smooth low-latency communication.

Features

High-quality video calling
Mute/unmute, camera on/off
Quick room join
Secure with appID & appSign
